# KukScript
Version Beta 0.2
KukScript is a Primitive Language Based off of C++ with slightly differing syntax. 

 This is a Very Basic Beta, So Supported Features are Few. 
 Currently Supports:                                       
 Printing (in the form of Output)                           
 Unique syntax (variables are defined: def name: type = val) ( i mean ts uses this too but this is different i swear)
 Variables
